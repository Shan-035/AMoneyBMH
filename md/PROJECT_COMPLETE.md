## 🚀 開始使用

### 方法一：直接使用
1. 雙擊 `start.bat` 啟動
2. 選擇「使用預設瀏覽器直接開啟」
3. 按照畫面指示進行設定

### 方法二：本地伺服器
1. 確保已安裝 Python 3
2. 雙擊 `start.bat`
3. 選擇「啟動本地伺服器」
4. 瀏覽器開啟 `http://localhost:8000`

### 方法三：手動開啟
1. 直接用瀏覽器開啟 `index.html`
2. 前往設定頁面配置所需的 API

## 🔑 必要設定 (5分鐘完成)

### 1. Google Sheets 設定 (必要)
- 建立新的 Google 試算表
- 複製 `google-sheets-setup.md` 中的 Apps Script 代碼
- 部署為 Web App 並取得 URL

### 2. API 金鑰設定 (可選)
- Alpha Vantage: 美股價格 (免費100次/日)
- CoinGecko: 加密貨幣價格 (免費使用)
- OpenAI: AI 分析功能 (按使用計費)

### 3. 初始資料輸入
- 新增現有持股的交易記錄
- 系統會自動計算持股數量和平均成本
- 價格會自動從 API 更新
