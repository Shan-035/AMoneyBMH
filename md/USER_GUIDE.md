# 投資組合管理系統使用說明

## 快速開始

### 1. 環境準備
- 任何現代網頁瀏覽器 (Chrome、Firefox、Safari、Edge)
- 穩定的網路連線
- Google 帳號 (用於 Google Sheets 整合)

### 2. 檔案部署
將所有檔案放置於 Web 伺服器目錄，或直接用瀏覽器開啟 `index.html`

### 3. 初始設定
1. 開啟系統後，點擊右上角「設定」
2. 依照需求設定各種 API 金鑰
3. 設定 Google Sheets 連接 (參考 `google-sheets-setup.md`)

## 主要功能介紹

### 📊 首頁 - 投資總覽
- **總資產統計**：顯示總資產、未實現損益、股票/加密貨幣資產分布
- **資產趨勢圖**：提供 7日/30日/90日/1年的資產變化趨勢
- **資產分布圓餅圖**：可切換檢視類型/幣種分布
- **持股明細表**：完整的持股資訊，支援搜尋和篩選

#### 使用技巧
- 點擊右上角貨幣選擇器可切換 TWD/USD/USDT 顯示
- 使用搜尋框快速找到特定股票
- 篩選功能可按市場類型分類檢視

### 💹 交易紀錄
- **新增交易**：支援買入/賣出交易記錄
- **交易篩選**：按日期、類型、市場、關鍵字篩選
- **成本計算**：自動計算 FIFO、LIFO、加權平均成本
- **來源標註**：可標記交易來源 (證券商、Binance、Pionex等)

#### 新增交易步驟
1. 點擊「新增交易」按鈕
2. 選擇交易類型 (買入/賣出)
3. 輸入股票代碼 (如: 2330, AAPL, BTC)
4. 填寫數量、價格、日期
5. 選擇交易來源並可添加備註
6. 點擊「新增」完成

### 📈 資產配置分析
- **AI 投資組合分析**：智能分析投資組合表現
- **風險評估**：自動評估投資風險並給出建議
- **配置圖表**：股票vs加密貨幣、幣種分布、產業分析
- **個股分析**：點擊個別股票可獲得 AI 深度分析

#### AI 分析功能
- 需要設定 OpenAI API Key 才能使用
- 提供投資組合整體評分 (0-100分)
- 給出具體的投資建議和風險提醒
- 支援個股深度分析

### ⚙️ 系統設定
- **Google Sheets 配置**：設定試算表連接
- **API 金鑰管理**：設定各種財經 API 和 AI API
- **一般設定**：預設貨幣、成本計算方式、自動更新等

## API 設定指南

### 必要 API (免費)
1. **Google Sheets**：用於資料儲存同步
2. **台股價格**：使用台灣證券交易所免費 API
3. **加密貨幣價格**：使用 CoinGecko 免費 API

### 選用 API (付費或有限制)
1. **Alpha Vantage** (美股)：每日 100 次免費查詢
2. **ExchangeRate API** (匯率)：每月 1000 次免費
3. **OpenAI** (AI 分析)：按使用量計費

### API 金鑰取得方式

#### Alpha Vantage (美股價格)
1. 前往 [alphavantage.co](https://www.alphavantage.co/support/#api-key)
2. 註冊免費帳號
3. 取得 API Key

#### CoinGecko (加密貨幣價格)
1. 前往 [coingecko.com](https://www.coingecko.com/en/api)
2. 免費版無需註冊，付費版提供更高頻率

#### OpenAI (AI 分析)
1. 前往 [platform.openai.com](https://platform.openai.com)
2. 註冊帳號並儲值
3. 在 API Keys 頁面建立新的金鑰

#### ExchangeRate API (匯率)
1. 前往 [exchangerate-api.com](https://app.exchangerate-api.com/sign-up)
2. 註冊免費帳號
3. 取得 API Key

## 使用流程建議

### 新使用者
1. **設定 Google Sheets**：按照 `google-sheets-setup.md` 建立資料表
2. **輸入現有持股**：在交易紀錄中手動新增過往交易
3. **設定 API**：至少設定基本的價格 API
4. **檢視分析**：查看資產配置和 AI 分析建議

### 日常使用
1. **記錄交易**：每次買賣後立即輸入交易紀錄
2. **檢查更新**：定期查看價格更新和損益變化
3. **分析調整**：參考 AI 分析調整投資策略
4. **風險監控**：留意風險評估和系統建議

## 進階功能

### 資料匯出匯入
- 支援 CSV 和 JSON 格式匯出
- 可用於資料備份或在其他工具中分析
- 支援從備份檔案還原資料

- `Ctrl + F`：快速搜尋功能
- `Ctrl + R`：重新整理價格資料

### 客製化設定
- 調整預設貨幣單位
- 選擇成本計算方式 (FIFO/LIFO/加權平均)
- 設定自動更新頻率

## 疑難排解

### 常見問題

#### Q: 價格無法更新？
A: 
1. 檢查網路連線
2. 確認 API Key 設定正確
3. 查看 API 使用額度是否用盡
4. 嘗試重新整理頁面

#### Q: Google Sheets 同步失敗？
A: 
1. 確認 Apps Script URL 正確
2. 檢查 Google Sheets 權限設定
3. 確認工作表名稱正確 (Holdings, Transactions)
4. 檢查網路防火牆設定

#### Q: AI 分析無法使用？
A: 
1. 確認已設定 OpenAI API Key
2. 檢查 API Key 餘額
3. 確認網路可以存取 OpenAI 服務

#### Q: 數據顯示錯誤？
A: 
1. 檢查輸入的交易記錄是否正確
2. 確認股票代碼格式正確
3. 嘗試清除瀏覽器快取
4. 重新載入頁面
